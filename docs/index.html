This is a screen test page. Cycle through each test screen with the following keys:

  • left arrow: previous page
  • enter: toggle fullscreen
  • down arrow: return to this screen
  • right arrow: next page

If you are on a touch screen without a keyboard touching the four quadrants will achieve the same result

  • left quadrant (white): previous page
  • middle top quandrant (red): toggle fullscreen
  • middle bottom quadrant (green): return to this screen
  • right quadrant (blue): next page
